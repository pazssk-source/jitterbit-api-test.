## Teste da API

Abaixo está um exemplo de requisição realizada utilizando o Postman para criação de um pedido na API.

A requisição utiliza o endpoint `POST /order` enviando os dados do pedido e seus respectivos itens no corpo da requisição em formato JSON.

A resposta da API confirma a criação do pedido juntamente com os itens associados no banco de dados.
